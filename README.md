# cc-company

Claude Code で仮想会社組織を構築・運営するプラグイン。

`/company` を実行すると秘書が窓口となり、あなたの事業に最適化された組織（部署）を構築。日常の運営では、秘書に話しかけるだけで CEO が適切な部署に仕事を振り分けます。

## インストール

```
/plugin marketplace add Shin-sibainu/cc-company
/plugin install company@cc-company
```

## コンセプト

```
あなた → 秘書（窓口） → CEO（振り分け） → 各部署
```

- **秘書**: 常に窓口。何でも相談OK。TODO管理、壁打ち、メモ
- **CEO**: 裏方で判断。部署が必要な案件を自動振り分け
- **各部署**: 専門領域のファイル管理を担当

ユーザーは部署を意識する必要なし。秘書に話しかけるだけ。

## 使い方

### 初回セットアップ

```
> /company

秘書: はじめまして！あなたの秘書になります。
      事業や活動について教えてください。
あなた: フリーランスのWeb開発やってます

秘書: 今の目標を教えてください。
あなた: SaaSを作って月10万の収益を目指してる

秘書: 日々困ってることはありますか？
あなた: タスクが散らかる、アイデアを忘れる、クライアント管理が雑

秘書: おすすめの部署構成はこちらです:
      秘書室, PM, リサーチ, 開発, 営業
      いかがですか？
あなた: OK

→ .company/ フォルダが自動生成される
```

### 日常の運営

```
> /company
秘書: おはようございます！何かありますか？

> LP作りたいんだよね、新しいサービスの

秘書: 承知しました。各部署に指示を出しますね。
  → PM: プロジェクト「新規LP」を作成
  → マーケ: ターゲット・訴求ポイントの整理
  → 開発: 技術要件の洗い出し

> 今日やること教えて

秘書: 今日のTODOです:
  最優先:
  - [ ] クライアントAへ見積もり送付
  通常:
  - [ ] LP設計書のレビュー
  - [ ] ブログ記事の下書き

> ちょっと壁打ちしたいんだけど、料金体系で悩んでて

秘書: いいですね！聞かせてください。
      今考えている選択肢はありますか？
```

## 部署一覧

| 部署 | 担当領域 | 常設 |
|------|---------|------|
| 秘書室 | TODO管理、壁打ち、メモ、相談 | はい |
| CEO | 意思決定、部署振り分け | はい |
| レビュー | 週次・月次レビュー | はい |
| PM | プロジェクト進捗、チケット管理 | |
| リサーチ | 市場調査、競合分析、技術調査 | |
| マーケティング | コンテンツ企画、SNS、キャンペーン | |
| 開発 | 技術ドキュメント、設計、デバッグ | |
| 経理 | 請求書、経費、売上管理 | |
| 営業 | クライアント管理、提案書 | |
| クリエイティブ | デザインブリーフ、ブランド管理 | |
| 人事 | 採用管理、チーム管理 | |
