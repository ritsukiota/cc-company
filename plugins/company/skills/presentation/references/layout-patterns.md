# レイアウトパターン

## 基本レイアウト構造

すべてのスライド（title, section-divider を除く）は以下の共通構造を持つ:

```
┌─────────────────────────────────────┐
│ [タイトルバー: 28pt bold]            │  0.8in
│ ─────────────────────────── accent線  │
│                                      │
│ [コンテンツエリア]                    │  5.5in
│                                      │
│                                      │
│ [フッター: ページ番号]               │  0.5in
└─────────────────────────────────────┘
```

---

## カラムレイアウト

### 2カラム（65:35）
- エグゼクティブサマリー、市場分析（テーブル+インサイト）に使用
- 左: 7.6in, 右: 3.9in, ガター: 0.3in

### 2カラム（50:50）
- 比較、Before/After に使用
- 左: 5.7in, 右: 5.7in, ガター: 0.3in

### 3カラム（均等）
- メトリクスカード並び
- 各列: 3.7in, ガター: 0.3in

### 4カラム（均等）
- メトリクスカード4つ並び
- 各列: 2.7in, ガター: 0.3in

## グリッドパターン

### 2x2 グリッド（SWOT用）
```
┌────────┬────────┐
│ TL     │ TR     │
│ 5.7in  │ 5.7in  │
│ 2.5in  │ 2.5in  │
├────────┼────────┤
│ BL     │ BR     │
│ 5.7in  │ 5.7in  │
│ 2.5in  │ 2.5in  │
└────────┴────────┘
ガター: 0.15in
```

---

## シェブロン/矢印パターン（ロードマップ用）

### 横並びシェブロン（3-5個）
```
  期間      期間      期間
┌──▶│──▶│──▶┐
│Phase1│Phase2│Phase3│
└──────┴──────┴──────┘
 ・MS1   ・MS1   ・MS1
 ・MS2   ・MS2   ・MS2
```
- シェブロン高さ: 1.2in
- 幅: (コンテンツ幅 - ガター合計) / フェーズ数
- マイルストーン: シェブロン下に箇条書き

---

## ピラミッドパターン

### 3-5段ピラミッド
```
      /\
     /  \
    /    \     ← 最上位（幅小）
   /──────\
  /        \   ← 中間
 /──────────\
/            \  ← 基盤（幅大）
```
- 各段: 台形で表現
- 上段ほど幅が狭い
- 段間ガター: 0.1in
- 右側に説明テキスト配置

---

## テーブルパターン

### スタンダードテーブル
- ヘッダー行: primary背景, 白文字, bold
- データ行: 交互にbackground / backgroundAlt
- ボーダー: border色, 1pt
- セルパディング: 0.1in
- フォント: ヘッダー12pt bold, データ11pt regular

### ハイライトテーブル（競合比較用）
- 自社行: accent1の薄い背景（opacity 10%）
- 優位項目: accent2色のテキスト
- 劣位項目: negative色のテキスト

---

## メトリクスカードパターン

### ヒーロー数値カード
```
┌──────────────────┐
│                   │
│      3.2兆円      │  48pt bold, accent1
│     市場規模       │  14pt, textMuted
│     ↑15% YoY      │  12pt, positive
│                   │
└──────────────────┘
```
- 背景: backgroundAlt
- 角丸: 0.1in
- パディング: 0.3in

---

## 優先度バッジ

### 提言スライド用
- High: `●` + 赤背景バッジ (#EF4444)
- Medium: `●` + 黄背景バッジ (#F59E0B)
- Low: `●` + 青背景バッジ (#3B82F6)
- サイズ: 8pt, 白文字, 角丸

---

## インサイトバー

### 市場分析・データスライド上部
```
┌─────────────────────────────────────┐
│ 💡 [インサイトテキスト]              │
└─────────────────────────────────────┘
```
- 背景: accent1 (15% opacity)
- テキスト: accent1色, 12pt bold
- 高さ: 0.5in
- 角丸: 0.05in

---

## マスタースライド

### 通常スライド（title, section-divider 以外）
- 背景: background色（白）
- 下部: 2pt の accent1 ライン
- 右下: ページ番号（10pt, textMuted）

### タイトルスライド
- 背景: primary色のグラデーション（左: primary, 右: secondary）
- テキスト: 白

### セクション区切り
- 背景: primary色（単色）
- テキスト: 白, 中央寄せ
